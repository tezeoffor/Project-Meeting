PROJECT MEETING APPLICATION

@Version 1

This is my implementation for the project planning apps that help you plan meetings.

With this app you will be able to :

- A user can create a meeting
- A user can set date and time by clicking on SET DATE/SET TIME

- A user can add number of attendees by clicking on ADD ATTENDEES button this takes you to a page that you will be to add new attendees

- A user will be able to add meeting location by clicking on the location button and       selecting your location from GoogleMaps

- When user create meetings, the will be able to view it fro the home page
